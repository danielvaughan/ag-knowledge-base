# Antigravity 2.0 Transition & Roadmap

This note tracks the evolution from Gemini CLI to the Antigravity 2.0 platform (IDE, SDK, and CLI).

## Recent Developments

- **IDE Integration (v0.7.0):** A specification for building Gemini CLI IDE plugins has been published, enabling rich, context-aware experiences and native in-editor diffing.
- **A2A Development (v0.5.0):** Introduction of the Agent2Agent (A2A) powered extension for advanced developer tool usage.
- **Cloud & Security (v0.4.0):** Automation of app deployment and security analysis via `/deploy` and `/security:analyze` commands.

## Platform Components

- **IDE:** Moving towards a plugin-based architecture for deep editor integration.
- **SDK:** Expanding capabilities for specialized agentic workflows (e.g., A2A).
- **CLI:** Transitioning from a standalone tool to a core component of the broader Antigravity ecosystem.
