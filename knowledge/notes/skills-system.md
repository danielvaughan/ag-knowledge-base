# Skills System (agent-tui, tui-tester)

The Skills system is a new framework introduced in the `v0.44.0-nightly` release to enhance agentic workflows and user interface capabilities.

## Overview

The system provides specialized tools for terminal-based agent interactions, primarily through `agent-tui` and `tui-tester`.

## Components

- **agent-tui**: A terminal user interface designed for interactive agent sessions. It allows users to visualize and interact with agent processes directly in the terminal.
- **tui-tester**: A testing utility designed to validate the behavior and responsiveness of the terminal user interface.

## Engineering Context

The introduction of these components signals a shift towards more sophisticated, interactive terminal-based environments for Antigravity 2.0, moving beyond simple CLI commands to a structured UI experience.
