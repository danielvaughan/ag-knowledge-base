# External Editor Support

The `v0.44.0-nightly` release introduces improved support for external text editors, enhancing the developer experience within the Antigravity 2.0 ecosystem.

## Supported Editors

- **Sublime Text**: Improved integration with error messaging and workflow handling.
- **Emacs Client**: Enhanced support for seamless interaction within Emacs-based development environments.

## Implementation Details

The support includes improved error reporting when interacting with these editors, ensuring that developers receive clearer feedback during the code-editing phase of an agentic workflow. This is part of a broader effort to make the Antigravity 2.0 environment more robust and developer-friendly.
