# Antigravity 2.0

Antigravity 2.0 is Google's next-generation agentic development
platform. It is designed to provide a seamless, AI-driven software
engineering experience through an integrated ecosystem.

## Overview

The platform comprises three core components:

- **IDE**: An intelligent development environment tailored for
  agentic workflows.
- **SDK**: A robust software development kit for building and
  extending platform capabilities.
- **CLI (Gemini CLI)**: A powerful command-line interface that
  serves as the primary touchpoint for developers to interact with
  Gemini models and automate complex engineering tasks.

## Relationship to Gemini CLI

The Gemini CLI is the current primary interface for interacting
with the Antigravity 2.0 ecosystem. It facilitates advanced
workflows such as file system operations, shell execution, and
web searching, all powered by Gemini models.
