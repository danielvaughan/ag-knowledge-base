## Summary

This research cycle addressed two significant gaps:

1. **ADK 2.0 GA** — A brand-new major release (May 19, 2026)
  introducing graph-based workflows, dynamic iterative workflows,
  and collaborative multi-agent workflows. ADK Python 2.0.0 GA is
  now available alongside Python 1.x, TypeScript, Go, Java, and
  Kotlin. The architecture introduces key primitives: Agent, Tool,
  Callbacks, Session, Memory, Artifact, Code Execution, Planning,
  Models, Event, and Runner.

2. **Google Cloud Data Platform Extensions** — The v0.6.0 changelog
  documents a comprehensive suite of Google Cloud data platform
  extensions for Gemini CLI including AlloyDB, BigQuery, Cloud SQL,
  Dataplex, Firestore, Looker, MySQL, Postgres, Spanner, SQL
  Server, and the MCP Toolbox. These were announced in a Google
  Cloud blog post on 2025-09-24 by Sujatha Mandava and Sandeep
  Karmarkar.

No new releases since v0.43.0-preview.1 (2026-05-19).
